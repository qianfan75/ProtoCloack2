# Security Policy

## Supported Versions

Only the latest public binary release is supported for security fixes unless a
release note states otherwise.

## Reporting a Vulnerability

Report vulnerabilities privately by email to **qianfan75@proton.me**. Do not
open a public GitHub issue for security-impacting findings.

Expected first response target: 7 days.

Please include version, operating system, deployment mode, relevant logs, and a
minimal reproduction when possible.
